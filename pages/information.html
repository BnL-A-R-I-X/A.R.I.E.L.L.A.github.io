[OVERVIEW]
BnL robotics product identifiers follow a standard
naming convention designed for both industrial
cataloguing and consumer-friendly branding.

[FORMAT]
[ACRONYM] – [CLASS LETTER / MODEL VARIANT]
• Acronym represents the robot's primary function.
• Class Letter designates production class, purpose,
  or deployment environment.
• Optional SERIAL NUMBER appended for tracking.
• Middle dot "·" used as corporate branding flourish.

[PRIMARY NAMING PATTERN]
ACRONYM: Derived from key words describing robot role.
  Example: WALL = Waste Allocation Load Lifter
CLASS LETTER: Single uppercase letter for environment/type.
  Example: E = Earth-Class, A = Axiom-Class
SUFFIX: Optional; T = Type designation; E = Engineering series.
HYBRID BRANDING: Names are constructed to be pronounceable
  and appealing for end users (WALL·E = "Wally").

[OBSERVATIONS]
1. Acronym always communicates the intended job function.
2. Class Letter indicates platform, deployment site, or role variant.
3. Hyphen separates model identifier from type/class letter.
4. Middle dot "·" is stylistic branding; not functionally required.
5. Naming scheme is consistent across all Buy n Large robotics products.
